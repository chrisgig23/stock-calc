Import Data — {{ account.account_name }}

Import your holdings and transaction history directly from a Schwab CSV export. Start with Step 1 to seed your current positions, then optionally add transaction history in Step 2 for full analytics.

Step 1 — Recommended First

Import Current Positions

Imports your current holdings, share counts, and total cost basis — even for shares purchased before the transaction history window.

How to export from Schwab:

  1. Log in to schwab.com and go to Accounts
  2. Click the Positions tab
  3. Click Export (top-right of the positions table)
  4. Save the downloaded .csv file — it will be named something like Personal-Positions-2026-03-26-211018.csv
  5. Select that file below and click Import Positions
Tip: You can re-import anytime to refresh quantities and cost basis. Existing holdings will be updated; nothing will be deleted.

Step 2 — Optional

Import Transaction History

Adds buy/sell history, dividends, and cash transfers — used to power analytics like dividend income, growth charts, and total return calculations. Schwab exports up to several years of history.

How to export from Schwab:

  1. Log in to schwab.com and go to Accounts
  2. Click the History tab
  3. Set the date range as far back as available (Schwab typically goes back 2–4 years)
  4. Click Export — the file will be named something like Personal_XXX703_Transactions_20260326-205510.csv
  5. Select that file below and click Import Transactions
Tip: You can re-import safely — duplicates are automatically detected and skipped so you won't get double entries.
